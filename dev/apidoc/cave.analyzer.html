cave.analyzer module¶

class cave.analyzer.Analyzer(original_rh, validated_rh, best_run, train_test, scenario, validator, pimp, model, output, max_pimp_samples, fanova_pairwise=True, rng=None)[source]¶

Bases: object

This class serves as an interface to all the individual analyzing and plotting components. The plotter object is responsible for the actual plotting of things, but should not be invoked via the facade (which is constructed for cmdline-usage).

Parameters:
  • original_rh (RunHistory) – runhistory containing all runs that have actually been run
  • validated_rh (RunHistory) – runhistory containing all runs from original_rh + estimates for default and all incumbents for all instances
  • incumbent (default,) – default and overall incumbent
  • train_test (bool) – whether is distinction is made (in cdf and scatter)
  • scenario (Scenario) – the scenario object
  • validator (Validator) – validator object (to estimate using EPM)
  • pimp (Importance) – parameter importance object with trained model
  • model (RandomForest) – random forest trained on original (combinated) runhistory
  • output (string) – output-directory
