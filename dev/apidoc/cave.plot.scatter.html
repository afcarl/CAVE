cave.plot.scatter module¶

cave.plot.scatter.plot_scatter_plot(x_data, y_data, labels, title='', debug=False, min_val=None, max_val=1000, grey_factor=1, linefactors=None, user_fontsize=20, dpi=100, metric='runtime', jitter_timeout=False, markers=None, sizes=None)[source]¶

method to generate a scatter plot :param x_data: numpy.array

performance values of one algorithm
Parameters:
  • y_data – numpy.array performance values of the other algorithm
  • labels – tuple (xlabel, ylabel)
  • title – str title of plot
  • debug – bool some debug options
  • min_val – float minimal value to plot
  • max_val – float maximal value to plot
  • grey_factor – float grey factor of points with a speedup of less 2
  • linefactors – list of floats factors of speedups
  • user_fontsize – int font size
  • dpi – int resolution
  • metric – str “runtime” or something else
  • jitter_timeout – bool Add some noise to remove timeout clutter
